之前布置的作业终于写完了，当我想要上传的时候，开始思考一个问题，如何把我新增的部分代码补充到之前上传代码的仓库中而且之后自己也能清晰地看出这是更完整的版本呢？经过一番探索，我掌握了将新增的代码上传到已有的github仓库中的方法。

##操作步骤

1. 打开git bash 命令行

2. git clone 到想要上传的仓库地址

3. 将自己要上传的内容复制到本地github文件夹下

4. 返回命令行——切换到克隆的文件夹下

5. 新建一个分支——git branch developmemt(我新建的分支命名为development)

6. 提交本地代码到github的folk上

   + git add
   
   + git commit -m"添加注释”
   
   + git push

完成以上步骤，就成功将更新后的代码提交到之前已经建立好的库中了，此时的代码在新建立的分支——development（自己随意命名）下面。
